# UI Guidelines

Framework:

- Tailwind CSS v4
- shadcn-vue

Theme:

- Professional
- Clean
- Minimal

Components:

- DataTable
- Card
- Dialog
- Drawer
- Sheet
- Dropdown Menu
- Calendar
- Date Picker

Dashboard:

Cards

Charts

Tables

Recent activity

All pages must:

- support dark mode
- support mobile devices
- support tablet devices

Forms:

- use validation
- show loading states
- show success messages
- show error messages
