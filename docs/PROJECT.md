# Rentora

Rentora is a property management platform built using:

- Laravel 13
- Vue 3
- TypeScript
- Inertia.js
- Tailwind CSS
- Vite

Users:

1. Owner
2. Tenant
3. Staff

Property types:

- Apartment
- Building
- House
- Boarding House

Features:

- Property management
- Unit management
- Tenant portal
- Billing
- Lease agreements
- Maintenance requests
- Notifications
- Reports

Goal:

Build maintainable, modular, scalable code suitable for SaaS.
